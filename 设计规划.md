# 学生信息管理系统 - 设计规划

## 一、项目背景

### 需求概述
开发一个基于 Java8 + SQLite + Swing 的学生信息管理系统，提供图形化界面操作，用于管理学生、教师、课程、成绩和考勤等信息。

### 技术选型
- **开发语言**: Java 8
- **数据库**: SQLite
- **界面框架**: Swing
- **项目结构**: Maven 标准结构
- **测试框架**: JUnit 5 + AssertJ
- **导出工具**: Apache POI 5.2.3

### 3.7 SQL 建表语句
```sql
-- 学生表
CREATE TABLE IF NOT EXISTS students (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    student_id VARCHAR(20) UNIQUE NOT NULL,
    name VARCHAR(50) NOT NULL,
    gender VARCHAR(10),
    age INTEGER,
    class_name VARCHAR(50),
    phone VARCHAR(20),
    email VARCHAR(100),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
CREATE INDEX IF NOT EXISTS idx_student_id ON students(student_id);
CREATE INDEX IF NOT EXISTS idx_name ON students(name);

-- 教师表
CREATE TABLE IF NOT EXISTS teachers (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    teacher_id VARCHAR(20) UNIQUE NOT NULL,
    name VARCHAR(50) NOT NULL,
    gender VARCHAR(10),
    title VARCHAR(50),
    department VARCHAR(100),
    phone VARCHAR(20),
    email VARCHAR(100),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
CREATE INDEX IF NOT EXISTS idx_teacher_id ON teachers(teacher_id);
CREATE INDEX IF NOT EXISTS idx_teacher_name ON teachers(name);

-- 课程表
CREATE TABLE IF NOT EXISTS courses (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    course_id VARCHAR(20) UNIQUE NOT NULL,
    course_name VARCHAR(100) NOT NULL,
    teacher_id INTEGER,
    teacher_name VARCHAR(50),
    credits DECIMAL(3,1),
    class_hours INTEGER,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
CREATE INDEX IF NOT EXISTS idx_course_id ON courses(course_id);

-- 成绩表
CREATE TABLE IF NOT EXISTS scores (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    student_id INTEGER NOT NULL,
    student_name VARCHAR(50),
    course_id INTEGER NOT NULL,
    course_name VARCHAR(100),
    score DECIMAL(5,2),
    term VARCHAR(50),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
CREATE INDEX IF NOT EXISTS idx_score_student ON scores(student_id);
CREATE INDEX IF NOT EXISTS idx_score_course ON scores(course_id);

-- 考勤表
CREATE TABLE IF NOT EXISTS attendances (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    student_id INTEGER NOT NULL,
    student_name VARCHAR(50),
    course_id INTEGER NOT NULL,
    course_name VARCHAR(100),
    attendance_date DATE NOT NULL,
    status VARCHAR(10),
    remark VARCHAR(200),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
CREATE INDEX IF NOT EXISTS idx_attendance_student ON attendances(student_id);
CREATE INDEX IF NOT EXISTS idx_attendance_course ON attendances(course_id);
CREATE INDEX IF NOT EXISTS idx_attendance_date ON attendances(attendance_date);

-- 选课表
CREATE TABLE IF NOT EXISTS enrollments (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    student_id INTEGER NOT NULL,
    course_id INTEGER NOT NULL,
    enroll_date DATE,
    status VARCHAR(20),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
CREATE INDEX IF NOT EXISTS idx_enrollment_student ON enrollments(student_id);
CREATE INDEX IF NOT EXISTS idx_enrollment_course ON enrollments(course_id);
```

## 四、功能模块设计

### 4.1 核心功能

#### 学生管理
1. **学生列表展示**: 表格形式展示所有学生信息
2. **添加学生**: 通过表单对话框添加新学生
3. **修改学生**: 选择学生后修改信息
4. **删除学生**: 删除选中的学生记录
5. **搜索学生**: 按学号或姓名搜索
6. **数据分页**: 大数据量时分页显示

#### 教师管理
1. **教师列表展示**: 表格形式展示所有教师信息
2. **添加教师**: 通过表单对话框添加新教师
3. **修改教师**: 选择教师后修改信息
4. **删除教师**: 删除选中的教师记录
5. **搜索教师**: 按工号或姓名搜索

#### 课程管理
1. **课程列表展示**: 表格形式展示所有课程信息
2. **添加课程**: 通过表单对话框添加新课程
3. **修改课程**: 选择课程后修改信息
4. **删除课程**: 删除选中的课程记录
5. **搜索课程**: 按课程号或课程名称搜索

#### 成绩管理
1. **成绩列表展示**: 表格形式展示所有成绩信息
2. **添加成绩**: 通过表单对话框添加新成绩
3. **修改成绩**: 选择成绩后修改信息
4. **删除成绩**: 删除选中的成绩记录
5. **搜索成绩**: 按学生或课程搜索

#### 考勤管理
1. **考勤列表展示**: 表格形式展示所有考勤记录
2. **添加考勤**: 通过表单对话框添加新考勤
3. **修改考勤**: 选择考勤记录后修改信息
4. **删除考勤**: 删除选中的考勤记录
5. **搜索考勤**: 按学生/课程/日期范围搜索
6. **统计分析**: 出勤率统计显示

#### 数据导出
1. **导出学生列表到 Excel**
2. **导出教师列表到 Excel**
3. **导出课程列表到 Excel**
4. **导出成绩列表到 Excel**
5. **导出考勤列表到 Excel**

#### 数据验证
1. **学号/工号格式验证**: 6-20 位字母或数字
2. **邮箱格式验证**: 正则表达式验证
3. **电话号码格式验证**: 支持手机号和座机号
4. **年龄范围验证**: 1-150
5. **成绩范围验证**: 0-100
6. **学分范围验证**: 0-20
7. **课时范围验证**: 1-200
